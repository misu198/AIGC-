# 混合命令 /blend
---
`/blend`命令可以快速上传2至5张图片，并分析每张图片的理念和美学，将它们融合成一张新的图像。



`/blend` 与`/imagine` 使用多个图像提示相同，但界面经过优化，使其在移动设备更易用。

`/blend` 最多可以使用5张图片。如果要在提示中使用超过5张图片，请使用 `/imagine` 命令和 image prompts。

`/blend` 与纯文本 prompts 不兼容。 如果需要同时使用图像提示和文本提示，请使用 `/imagine` 命令和 image prompts。

## `/blend` 选项

输入 `/blend` 命令后，将提示您上传两张照片。当使用移动设备时，可拖放图像或从照片库添加图像。要添加更多图像，请选择可选/选项字段并选择image3、image4或image5。

`/blend` 命令可能比其他命令启动时间更长，因为在 Midjourney 机器人处理您的请求之前，必须先上传您的图片。



混合图像默认使用1:1长宽比，但您可以使用可选的 dimensions 字段在正方形长宽比（1:1）、竖向长宽比（2:3）或横向长宽比（3:2）之间进行选择。



自定义后缀被添加到/blend prompts 的末尾，就像其他 `/imagine prompts` 一样。作为`/blend `命令的一部分指定的长宽比会覆盖自定义后缀中的长宽比。
