# Midjourney 参数详解-Tile 平铺

---

`--tile` 参数生成可用作重复平铺的图像，以创建面料、壁纸和纹理的无缝（四方连续，就像拼瓷砖的花纹）图案。



`--tile`适用于模型版本1、2、3 和 5。

`--tile`仅生成单个平铺。使用类似[无缝图案检查器](https://www.pycheung.com/checker/)的图案制作工具来查看平铺效果。
