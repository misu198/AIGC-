# Midjourney做出参考图风格的绘画教程

![img](https://cdn.jsdelivr.net/gh/misu198/Midjourney@main/guge/c19974a95e47f481713448258.png_q900)

## 操作步骤：

### 1：切换模型，使用设置命令：

#### **使用/settings 并从菜单中选择版本：🍎 Niji Version 5**

![img](https://cdn.jsdelivr.net/gh/misu198/Midjourney@main/guge/195aa2121e7e1713448258.png_q900)



### 2：点击上传文件，添加参考图，获取URL地址。

![img](https://cdn.jsdelivr.net/gh/misu198/Midjourney@main/guge/71af70e889eb62e1713448267.png_q900)



### 3：图生文：使用/describe并上传参考图，获得4个描述图像的文本提示。

![img](https://cdn.jsdelivr.net/gh/misu198/Midjourney@main/guge/ea91c1e668e86521713448258.png_q900)



### 4：文生图：垫图使用参考图获取的URL+生成的图像文本提示（选择其一）+（参数）--iw 2 --ar 4:7

#### **--iw** 设置相对于文本权重的图像提示权重，默认值为 --iw 0.25。用于生成图像接近参考图。

![img](https://cdn.jsdelivr.net/gh/misu198/Midjourney@main/guge/9738fdc23b6a6ba1713448276.png_q900)



### 5：同风格图像男孩换成女孩，改动少量图像文本提示词：男孩换成很酷的时尚女孩，同时去掉图像提示权重 --iw 2

![img](https://cdn.jsdelivr.net/gh/misu198/Midjourney@main/guge/45dde6ef7b768bb1713448282.png_q900)
